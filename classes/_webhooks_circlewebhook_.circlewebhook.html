Private updateYamlDocument

  • updateYamlDocument(yml: any, original: string, url: string): string
    • Defined in webhooks/CircleWebhook.ts:82

    Add the webhook to a Yaml document if it doesn't already exist. It does preserve the existing structure of the notification webhook section if it already exists as well.

    Two calls are added:

    • one explicit call as soon as the build starts (with Circle 1, it happens after the checkout, with Circle 2, it happens as the first step of the build job)
    • one implicit at the end of the build job through the notification section

    Parameters

    • yml: any

      parsed yaml document

    • original: string
    • url: string

    Returns string
